# kkDanubius
Basketball Club Danubius webpage  

To run this app:  
1. go into danubius dir  
2. install dependencies :$ npm install  
3. then type :$ npm start  
4. open the browser and go to lockalhost:3000  
5. 

Using: 
1. Express  
2. Node.js  
3. AngularJS  
4. JavaScript  
5. HTML  
6. CSS  
7. jQuery  
8. Bootstrap  
9. Fency-Font
